# Hotel Booking Data Analysis

## Project Overview

This project analyzes hotel booking data from AtliQ Hotels to uncover meaningful business insights using Exploratory Data Analysis (EDA). The analysis focuses on booking trends, occupancy, revenue, cancellations, customer ratings, and city-wise performance.

## Objectives

- Analyze hotel booking patterns
- Understand occupancy trends
- Identify revenue performance
- Study booking cancellations
- Compare performance across cities and hotels
- Visualize insights using charts

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Files

- `project.ipynb` – Complete Jupyter Notebook containing data cleaning, analysis, and visualizations.
- `project.html` – HTML version of the notebook for easy viewing in a browser.

## Key Analysis Performed

- Data Cleaning

- Revenue Analysis
- Occupancy Analysis
- Booking Trends
- City-wise Analysis
- Visualizations using Matplotlib

## Skills Demonstrated
